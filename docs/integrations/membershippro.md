# Membership Pro

Joomdle integrates with [Membership Pro](https://extensions.joomla.org/extension/membership-pro/), a Joomla subscription and membership management extension. This integration links Moodle courses to membership plans, automating enrolment based on a user's subscription status.

## Available Plugins

* **Joomdle Membership Pro** — Enables the integration, automatically enrolling users in the configured Moodle courses when they subscribe to a plan.

## How It Works

- When a user purchases a membership plan, they are automatically enrolled in the Moodle courses associated with that plan.
- When the plan expires, their enrolment in those courses is automatically suspended.

## Configuration

Once the plugin is installed and enabled, a new **Joomdle settings** tab appears in each Plan configuration page in Membership Pro.

1. Go to the Membership Pro admin panel and open a plan.
2. Click the **Joomdle settings** tab.
3. Select the Moodle courses you want to associate with this plan.
4. Save the plan.

Repeat for each plan you want to link to Moodle courses. A single plan can be associated with multiple courses.
