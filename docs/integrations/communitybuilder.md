# Community Builder

Joomdle integrates with [Community Builder](https://www.joomlapolis.com/), a popular Joomla extension for user profile management. This integration allows Community Builder to act as an additional data source for user profile information that is synced to Moodle.

## Required Plugins

To use the Joomdle–Community Builder integration, you must install and enable the following plugins:

* **Joomdle Profile Community Builder plugin** — This plugin lets you use Community Builder as an additional data source for user profile information that is synced to Moodle.
* **Joomdle User Community Builder plugin** — Responds to Community Builder events by calling Joomdle user sync functions. This plugin must be installed through Community Builder's own plugin installer, not through the standard Joomla extension installer.

## Profile Plugins

Joomdle provides Community Builder profile tab plugins that display Moodle-related information on a user's CB profile page:

* **Community Builder My Courses** — Displays the courses the user is enrolled in on Moodle
* **Community Builder My Certificates** — Displays the certificates the user has earned in Moodle
* **Community Builder My Badges** — Displays the badges the user has earned in Moodle
