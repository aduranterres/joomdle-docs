# Installing Joomdle for Joomla

The **Joomdle for Joomla!** extension consists of a main component, along with several modules and plugins. In addition, many extra Joomdle packages are available for download from the Joomdle website.

To install Joomdle for Joomla!, use Joomla!’s **Extension Manager** and upload the package named **`pkg_joomdle.zip`**. This single package installs the component, plugins, and modules in one step, making the installation process quick and straightforward.

> **Note:** During installation, all plugins are enabled and the modules are created, but the modules are not published by default.
