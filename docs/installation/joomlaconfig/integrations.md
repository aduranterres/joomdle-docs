# Integrations

Configuration for other third-party integrations.

## Mailing lists

### Mailing lists integration

Select the mailing list component to integrate with. This allows Joomdle to manage mailing list subscriptions based on course enrolments.

## PDF

### Use PDF integration

When enabled, Joomdle can generate PDF documents (e.g. course certificates). Requires the TCPDF library to be installed.
