# Views

The Views section of the Joomdle configuration controls how courses and related content are displayed across the different views available in Joomdle.

## Action Buttons

These settings control which action buttons are shown when browsing courses.

| Option | Description |
|--------|-------------|
| Show button for free courses | Display an action button on free courses in detail and list views |
| Show button for paid courses | Display an action button on paid courses |
| Go to course button | Destination link for the course navigation button |

## Topics View

| Option | Description |
|--------|-------------|
| Show headings | Show or hide topic headings in the topics view |

## Course Category View

| Option | Description |
|--------|-------------|
| Show category info | Display category information in category listings |
| Show action buttons | Show action buttons in category listings |

## Back Links

| Option | Description |
|--------|-------------|
| Show back links | Show or hide back navigation links throughout the views |

## Detail View

The detail view has the most configuration options, allowing you to control exactly which course information is shown on the course detail page.

| Option | Description |
|--------|-------------|
| Show category | Display the course category |
| Show course summary | Display the course summary |
| Show language | Display the course language |
| Show start date | Display the course start date |
| Show enrol dates | Display enrolment start and end dates |
| Show enrol period | Display the enrolment period duration |
| Show number of topics | Display the number of topics in the course |
| Show cost | Display the course cost |
| Show contents link | Display a link to the course contents |
| Show topics link | Display a link to the course topics |
| Show grading system link | Display a link to the grading system |
| Show teachers link | Display a link to the course teachers |

## Courses ABC View

| Option | Description |
|--------|-------------|
| Show action buttons | Display action buttons in the alphabetical course listing |
