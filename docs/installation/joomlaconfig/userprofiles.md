# User profiles

Configuration for user profile data sources and Joomla user group mappings.

## Additional data source

### Additional data source

Select the extension to use as a profile data source. This allows Joomdle to pull additional user profile information from a third-party extension. You can add more options by downloading specific plugins from joomdle.com.

## Joomla user groups

### Teachers group

Joomla user group to use as the parent group for each course teachers group. When Joomdle creates per-course teacher groups, they will be nested under this group.

### Students group

Joomla user group to use as the parent group for each course students group. When Joomdle creates per-course student groups, they will be nested under this group.
