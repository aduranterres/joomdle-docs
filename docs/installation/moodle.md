# Installing Joomdle for Moodle

To enable Moodle to authenticate users through Joomla!, you need to install and configure the **Joomdle authentication plugin**. The plugin is included in the Joomdle installation package as **`auth_joomdle.zip`**, located in the `moodle/` folder.

You can install the plugin directly through the Moodle interface by going to:

**Site administration → Plugins → Install add-ons**

From there, upload the plugin package and follow the on-screen instructions to complete the installation. You will be redirected to the settings screen, where you will need to enter at least:
* Joomla URL
* Joomla auth token

You can see all the configuration options in the next section.
